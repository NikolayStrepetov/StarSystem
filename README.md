# StarSystem
Простой консольный генератор звёздной системы на языке C#.

## MySpace.cs
Файл содержит код меню приложения.

## StarSystem.cs
Описание класса StarSystem и его методов добавления и получения звезды и планет, а также очистка списка планет.

## Star.cs
Описание класса Star, его свойств и методов.

## Planet.cs
Абстрактный класс, описывающий планету, который наследуется следующими классами.

## Dwarf.cs ,  EarthType.cs ,  Giant.cs
Дочерние классы, наследующие методы класса Planet.

## Satelite.cs
Описание класса спутника планеты.

## IEarthLike.cs
Интерфейс, содержащий методы, которые реализуются классе EarthType.

## EarthlikeCollection.cs
Содержит структуру, использующую обобщения, а также метод сортировки планет в ней по коэффициенту землеподобия.

## Rocket.cs ,  SpaceShip.cs ,  FlyProgress.cs и IFlyProgress.cs
Нужны для демонстрации ковариантности и контрвариантности.

## ClassLogger.cs
Файл, описывающий ClassLogger, необходимый для записи лога.

## Error.cs
Файл, описывающий класс ошибки, предназначен для демонстрации работы делегатов.

## config.txt
Файл конфигурации.
